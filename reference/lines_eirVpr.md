# Add lines for the EIR(t) *vs.* the PR(t)

Add lines for the EIR(t) *vs.* the PR(t)

## Usage

``` r
lines_eirVpr(eir, pr, nStrata, clrs = "black", llty = 1)
```

## Arguments

- eir:

  the daily EIR

- pr:

  the parasite rate

- nStrata:

  the number of population strata

- clrs:

  a [character](https://rdrr.io/r/base/character.html) vector of colors

- llty:

  an [integer](https://rdrr.io/r/base/integer.html) (or integers) that
  specifies `lty` for xds_plotting
