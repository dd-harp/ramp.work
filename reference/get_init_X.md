# Get Initial Values for Parameters

Get Initial Values for Parameters

## Usage

``` r
get_init_X(xds_obj, feature, options = list())
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- feature:

  the dispatch string

- options:

  a list to configure feature-specific indexing

## Value

a list with the mean peak and the values
