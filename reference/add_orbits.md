# Draw the orbit for the \\i^{th}\\ element of eirpr\$scaling.

Draw the orbit for the \\i^{th}\\ element of eirpr\$scaling.

## Usage

``` r
add_orbits(i, xds_obj, clr)
```

## Arguments

- i:

  the index of the orbit to xds_plot

- xds_obj:

  a **`ramp.xds`** model object

- clr:

  a [character](https://rdrr.io/r/base/character.html) vector of colors
