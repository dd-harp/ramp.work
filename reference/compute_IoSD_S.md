# Compute the Index of Seasonal Dispersal for a seasonal pattern \\S(t)\\ over one year. This assumes that \$\$\int_0^{365} S(t)dt = 365\$\$

If in doubt, use
[compute_IoSD_F](https://dd-harp.github.io/ramp.work/reference/compute_IoSD_F.md)

## Usage

``` r
compute_IoSD_S(S)
```

## Arguments

- S:

  a function describing a seasonal pattern
