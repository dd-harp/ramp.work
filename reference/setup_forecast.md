# Set the forecast interpolation

Set the forecast interpolation

## Usage

``` r
setup_forecast(xds_obj, N = 0, method = "use_last")
```

## Arguments

- xds_obj:

  a **`ramp.xds`** xds_obj object

- N:

  the number of years to forecast

- method:

  to dispatch setup_forecast_y

## Value

a **`ramp.xds`** xds_obj object
