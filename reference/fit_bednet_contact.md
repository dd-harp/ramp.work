# Fit bed net coverage

As part of a dynamical time series analysis, fit the parameter
describing bed net coverage

## Usage

``` r
fit_bednet_contact(xds_obj, options = list())
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- options:

  setup options for the bed net rounds indexing

## Value

a **`ramp.xds`** model object
