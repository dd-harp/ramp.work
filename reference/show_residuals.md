# Plot the model and the data

Plot the model and the data

## Usage

``` r
show_residuals(xds_obj, clr = "black", add = FALSE)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- clr:

  the color of the line to be plotted

- add:

  add to an existing show_fit plot

## Value

the PR, invisibly
