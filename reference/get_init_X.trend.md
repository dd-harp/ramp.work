# Get Initial Values for Parameters

Get Initial Values for Parameters

## Usage

``` r
# S3 method for class 'trend'
get_init_X(xds_obj, feature, options)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- feature:

  the dispatch string

- options:

  a list to configure feature-specific indexing

## Value

a vector
