# Change Spline \\t\\ Values

Change Spline \\t\\ Values

## Usage

``` r
change_ix_fit_spline_t(new_t, t_ix, xds_obj)
```

## Arguments

- new_t:

  the new \\t\\ interpolation point value(s)

- t_ix:

  the indices of the values to replace

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
