# Fit IRS shock_size

As part of a dynamical time series analysis, fit the parameter(s)
describing the scale of IRS effect sizes.

This should not be used

## Usage

``` r
fit_irs_shock_size(xds_obj, options = list())
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- options:

  setup options for the irs rounds indexing

## Value

a **`ramp.xds`** model object
