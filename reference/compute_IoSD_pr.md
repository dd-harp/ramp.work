# Compute the Index of Seasonal Dispersal for a PR seasonal orbit for the \\i^{th}\\ element of `eirpr$scaling`

Compute the Index of Seasonal Dispersal for a PR seasonal orbit for the
\\i^{th}\\ element of `eirpr$scaling`

## Usage

``` r
compute_IoSD_pr(i, pars)
```

## Arguments

- i:

  the index of the orbit to be evaluated

- pars:

  an **`xds`** object
