# Set the forecast interpolation

Set the forecast interpolation

## Usage

``` r
forecast_ty(xds_obj, ix = c())
```

## Arguments

- xds_obj:

  a **`ramp.xds`** xds_obj object

- ix:

  (optional) indices for
  [impute_value](https://dd-harp.github.io/ramp.work/reference/impute_value.md)

## Value

a **`ramp.xds`** xds_obj object
