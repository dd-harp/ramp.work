# Update a function

Update a function

## Usage

``` r
update_function_X(X, xds_obj, feature, options = list())
```

## Arguments

- X:

  new parameter values

- xds_obj:

  a **`ramp.xds`** xds_obj object

- feature:

  a string to dispatch

- options:

  a list to configure feature-specific indexing

## Value

sum of squared differences
