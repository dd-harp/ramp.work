# Replace Spline \\y\\ Values

Replace Spline \\y\\ Values

## Usage

``` r
change_all_fit_spline_ty(yy, tt, xds_obj)
```

## Arguments

- yy:

  a new set of \\y\\ values for the interpolation points

- tt:

  a new set of \\t\\ values for the interpolation points

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
