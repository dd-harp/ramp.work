# Impute the baseline

Impute the \\n\\ value of one or more interpolation points in the pair,
\\t,y\\.

## Usage

``` r
impute_value(y, impute_y, N = 1)
```

## Arguments

- y:

  a set of y values

- impute_y:

  a text string to dispatch `impute_value`

- N:

  the length of the return value

## Value

numeric
