# Draw the orbit for the \\i^{th}\\ element of eirpr\$scaling, and add points at the minimum and maximum eir and pr

Draw the orbit for the \\i^{th}\\ element of eirpr\$scaling, and add
points at the minimum and maximum eir and pr

## Usage

``` r
add_orbits_px(i, xds_obj, clr)
```

## Arguments

- i:

  the index of the orbit to xds_plot

- xds_obj:

  a **`ramp.xds`** model object

- clr:

  a [character](https://rdrr.io/r/base/character.html) vector of colors
