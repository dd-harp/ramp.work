# Compute the GoF for `X`

Compute the GoF for `X`

## Usage

``` r
# S3 method for class 'multifit'
get_init_X(xds_obj, feature, options = list())
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- feature:

  the dispatch string

- options:

  a list to configure feature-specific indexing

## Value

sum of squared differences
