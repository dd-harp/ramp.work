# Setup indices for irs shock_d50

Setup indices for irs shock_d50

## Usage

``` r
# S3 method for class 'irs_d50'
setup_fitting_indices(xds_obj, feature, options)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- feature:

  a list of model features to feature

- options:

  options for setting indices

## Value

indices
