# Impute the baseline

Impute the a new \\y\\ value(s) for one or more interpolation points

## Usage

``` r
change_forecast_ty(xds_obj, t, y)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- t:

  new \\t\\-values for the forecast interpolation points

- y:

  new \\y\\-values for the forecast interpolation points

## Value

an **`ramp.xds`** model object
