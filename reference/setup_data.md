# Set up the data object for fitting

Set up the data object for fitting

## Usage

``` r
setup_data(xds_obj, pfpr, jdates, yr0 = 2015)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- pfpr:

  a *Pf*PR time series

- jdates:

  julian dates for `pfpr`

- yr0:

  the starting year

## Value

an **`xds_obj`**
