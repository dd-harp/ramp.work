# Get initial X: IRS shock

Get initial X: IRS shock

``` r
# S3 method for class 'irs_shock'
get_init_X(xds_obj, feature, options)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- feature:

  the dispatch string

- options:

  a list to configure feature-specific indexing

## Value

IRS shock levels
