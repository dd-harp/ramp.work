# Change Spline \\y\\ Values

Change Spline \\y\\ Values

## Usage

``` r
change_ix_fit_spline_y(new_y, y_ix, xds_obj)
```

## Arguments

- new_y:

  the new \\y\\ interpolation point value(s)

- y_ix:

  the indices of the values to replace

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
