# Using the eirpr matrix and a RM xds_obj, convert pr to Lambda

This takes a xds_obj and uses the XH component to define the eirpr
relationship using `xde_scaling_eir` then calls `xde_scaling_lambda`

## Usage

``` r
pr2Lambda(pr, xds_obj, constrain = TRUE)
```

## Arguments

- pr:

  a vector

- xds_obj:

  a list that defines an `ramp.xds` xds_obj (*e.g.*, generated by
  `xde_setup()`)

- constrain:

  a logical, if TRUE then set all negative values to zero
