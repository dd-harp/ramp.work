# Adjust Spacing

Adjust Spacing

## Usage

``` r
event_chop_spline_t(xds_obj, b4 = 30, fu = 210, mngp = 90, mxgp = 500)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- b4:

  days before intervention

- fu:

  days after intervention

- mngp:

  minimum gap

- mxgp:

  the maximum gap

## Value

a **`ramp.xds`** model object
