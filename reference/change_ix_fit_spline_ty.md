# Change Spline \\t,y\\ Values

Change Spline \\t,y\\ Values

## Usage

``` r
change_ix_fit_spline_ty(new_y, new_t, ix, xds_obj)
```

## Arguments

- new_y:

  the new \\y\\ interpolation point value(s)

- new_t:

  the new \\t\\ interpolation point value(s)

- ix:

  the indices of the values to replace

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
