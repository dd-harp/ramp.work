# Time Since Event

Compute the time elapsed since the last vector control event for all the
spline \\t\\-values

## Usage

``` r
get_yix_after_irs_round(round_ix, xds_obj, N = 1)
```

## Arguments

- round_ix:

  the index of the IRS round

- xds_obj:

  a **`ramp.xds`** model object

- N:

  the number of y indices

## Value

numeric
