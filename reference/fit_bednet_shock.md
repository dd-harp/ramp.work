# Fit Bednet Shock

As part of a dynamical time series analysis, fit the parameter(s)
describing the scale of bednet effect sizes.

This should not be used

## Usage

``` r
fit_bednet_shock(xds_obj, options = list())
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- options:

  setup options for the bednet rounds indexing

## Value

a **`ramp.xds`** model object
