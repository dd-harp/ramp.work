# Convert pr to ni

Use the scaling table to interpolate

## Usage

``` r
xds_pr2ni(pr, xds_obj, extend = FALSE)
```

## Arguments

- pr:

  a [vector](https://rdrr.io/r/base/vector.html)

- xds_obj:

  a [list](https://rdrr.io/r/base/list.html)

- extend:

  a [logical](https://rdrr.io/r/base/logical.html) option to determine
  whether to extend outside the range
