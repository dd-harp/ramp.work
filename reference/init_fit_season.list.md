# Set up the seasonal pattern

If the current `season_par` is a list, return a parameter set

## Usage

``` r
# S3 method for class 'list'
init_fit_season(season_par)
```

## Arguments

- season_par:

  seasonal pattern parameters

## Value

a `sin` function parameter set
