# Get limits for IRS coverage parameters

Get limits for IRS coverage parameters

## Usage

``` r
# S3 method for class 'bednet_contact'
get_limits_X(xds_obj, feature = "bednet_contact")
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- feature:

  the dispatch string

## Value

bounds on coverage, as [numeric](https://rdrr.io/r/base/numeric.html)
