# Set up the pre-observation interpolating points

Set up the pre-observation interpolating points

## Usage

``` r
# S3 method for class 'asis'
setup_hindcast_y(y, xds_obj)
```

## Arguments

- y:

  the y values for interpolation

- xds_obj:

  a **\`ramp.xds** xds_obj object

## Value

a **\`ramp.xds** xds_obj object

## Note

Set the \\y\\ value for pre-observation interpolation points to a
constant.
